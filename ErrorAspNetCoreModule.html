Error en aplicación .NET 5+ en IIS

Si encuentras un error al intentar ejecutar una aplicación .NET 5+ en IIS, es posible que el problema esté relacionado con la falta del AspNetCoreModule. Este módulo es necesario para que IIS pueda manejar las solicitudes para aplicaciones .NET Core y .NET 5+.

HTTP Error 500.19 - Internal Server Error

The requested page cannot be accessed because the related configuration data for the page is invalid.
Config Source: : 

-1:
0:

Solución

Debes instalar el .NET Core Hosting Bundle en tu servidor. Este paquete incluye el runtime de .NET, el runtime de ASP.NET Core y el AspNetCoreModule necesario para IIS.

Instalación

Ve al sitio oficial de descargas de .NET (https://dotnet.microsoft.com/download) y descarga el .NET Core Hosting Bundle para la versión de .NET que estás utilizando.

Una vez que lo hayas descargado, ejecuta el instalador en tu servidor y sigue las instrucciones en pantalla para completar la instalación.

Cuando la instalación se haya completado, reinicia IIS. Puedes hacerlo desde la línea de comandos con el comando: iisreset

Ahora IIS debería tener instalado el AspNetCoreModule y ser capaz de alojar aplicaciones ASP.NET Core y .NET 5+.
